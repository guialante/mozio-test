Basic Commands
--------------

Load Data
~~~~~~~~~

::

  $ ./manage.py loaddata providers.json



Running tests with py.test
~~~~~~~~~~~~~~~~~~~~~~~~~~

::

  $ py.test





Install Requirements
~~~~~~~~~~~~~~~~~~~~

::

  $ pip install -r requirements/local.txt
